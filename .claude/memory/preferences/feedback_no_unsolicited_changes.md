NEVER change dimensions, sizes, spacing, or any design value unless the operator explicitly asks for it.

**Why:** The operator asked to fix alignment and borders. Claude also changed header height from 44px to 40px without being asked. This wasted time and broke the design intent.

**How to apply:** When fixing a specific issue, ONLY fix that issue. Do not "improve" adjacent values. If a height, width, font size, spacing, or any visual measurement needs changing, wait for the operator to say so. Zero unsolicited design changes.
