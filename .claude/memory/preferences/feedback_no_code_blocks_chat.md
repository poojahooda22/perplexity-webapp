NEVER use code blocks or code fences in chat messages. No triple-backtick blocks, no scrollable code areas.

**Why:** Nobody reads scrollable code blocks in chat. They're not helpful. The user explicitly called this "dumb shit."

**How to apply:** If you need to share code, equations, or detailed plans — write them to a .md file and reference it. Keep chat responses as plain text with bold labels and numbered lists. Short, direct, no code fences.
